# Stavby T Martin - Webstránka

## Popis
Kompletná responzívna webstránka pre stavebnú firmu Stavby T Martin vytvorená podľa požiadaviek zákazníka. Stránka obsahuje moderný dizajn, smooth animácie a všetky požadované sekcie.

## Funkcie
- ✅ Plne responzívny dizajn
- ✅ Moderný čistý vzhľad v farbách #00a2e3
- ✅ Smooth scrolling a animácie
- ✅ Galéria s lightbox efektom
- ✅ Kontaktný formulár s mailto funkciou
- ✅ SEO-friendly štruktúra
- ✅ Mobilné menu
- ✅ Všetky texty zo stránky www.stavbytmartin.sk

## Sekcie
1. **Header** - Navigácia s logom
2. **Hero sekcia** - Úvod s CTA tlačidlom
3. **O nás** - Informácie o firme a prednosti
4. **Referencie** - Karty s realizovanými projektmi
5. **Galéria** - Fotogaléria s lightboxom
6. **Kontakt** - Kontaktné údaje a formulár
7. **Footer** - Základné informácie

## Použitie
1. Otvorte `index.html` vo webovom prehliadači
2. Pre mobilné zariadenia použite responzívny režim v prehliadači
3. V galérii kliknite na obrázky pre zväčšenie
4. Kontaktný formulár otvorí email klienta predvyplnený obsahom

## Prispôsobenie
- **Logo**: Nahraďte ikonu v headeri vlastným logom
- **Obrázky**: V galérii nahraďte placeholder obrázky skutočnými fotkami
- **Farby**: Zmeňte CSS premenné v :root pre zmenu farebnej schémy
- **Texty**: Upravte obsah podľa potreby

## Technické informácie
- Čistý HTML5, CSS3 a JavaScript
- Žiadne externé frameworky
- Google Fonts (Inter + Playfair Display)
- Font Awesome ikony
- Responzívny dizajn (mobile-first)
